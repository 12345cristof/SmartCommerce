# SmartCommerce

Sistema de gestión de comercio electrónico desarrollado en Go.

## Descripción

SmartCommerce es una plataforma web para la gestión de usuarios, productos, inventario, pedidos y reportes dentro de una tienda virtual.

## Módulos

### Gestión de Usuarios
- Registro de usuarios
- Inicio de sesión
- Recuperación de contraseñas
- Gestión de perfiles

### Gestión de Productos
- Registro de productos
- Edición de productos
- Gestión de categorías
- Gestión de precios

### Gestión de Inventario
- Control de stock
- Alertas de inventario

### Gestión de Pedidos
- Creación de pedidos
- Historial de compras
- Actualización de estados

### Gestión de Reportes
- Reportes de ventas
- Reportes de clientes
- Reportes de productos
- Reportes de inventario

## Tecnologías

- Go
- Gin Framework
- PostgreSQL
- JWT
- GitHub
